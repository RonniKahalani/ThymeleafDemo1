Spring & Thymeleaf

This app demonstrates how the MVC pattern is used in Spring with Thymeleaf and Bootstrap, using a simple user form for data entry and a list of users.

Multi Language Support

This feature uses multi-language support, which is managed via an url parameter, called lang.

The texts are defined in messages_ <language code>.properties files in the src/main/resources project folder.

When selecting a language from the dropdown below, the page is updated and showing the welcome message in the selected language.

Notice the browser address line, you can see the added url param ?lang=<language code>.

This feature is just a simple demo, in real live all texts, in the app, would be internationalized into these messages files.

The Business Logic & Data

Includes services-, repositories- and entity classes.

Services are business logic

Handles business logic and access data via repositories.

Repositories encapsulate and supply data

Handles CRUD: Create, Read, Update, Delete and querying data.

The entity is the data

Represents a database object (a row in a database table).

The User Interface (UI)

Includes HTML-, CSS- and JavaScript files.

HTML structures a view

Defines the structure of event-driven elements on a web page.

CSS styles a view

Defines the look and feel style of the page elements.

JavaScript makes views dynamic

Manipulates view structure, data, style and user interaction.

The Request Delegator

Gets incoming web requests from views/users and response with a result view.

Delegates requests from a view

Validates and delegates the requests to the targeted service.

Respond with data from services

Returns a view and including data from the service.

Controls all communication

Can convert data on the fly, because it controls all in- and outgoing data.

The purpose of this app is to serve as a example app in computer science classes, showing what a Spring-, Thymeleaf- and Bootstrap-based app can look like.

The source code is available on GitHub at: https://github.com/RonniKahalani/ThymeleafDemo1

Using features
  • Controllers (HomeController and UserController).
  • Model components like services, repositories and data entities.
  • Thymeleaf v3.5.5 with HTML fragment templates, view code is written only once and reused.
  • UI framework Bootstrap v5.3.3 with jQuery v1.7.3.
  • Bootstrap Icons v1.10.5 + Font Awesome Icons v6.0.0.
  • Local CSS and JavaScript files.
  • Using media audio and video.
  • Custom exception handler.
  • Implementing the Spring Controller, Service, Repository and Entity concepts.
  • Shows a feature using multi-language support (i18n), via locale specific messages_xx.properties files.
Controller endpoints
  • Home - HTTP GET /
  • List Users - HTTP GET - /user
  • Show Create User Form - HTTP GET /create
  • Save User - HTTP POST - /user
  • Show User - HTTP GET - /user/{id}
  • Show Update User Form - HTTP GET /update/{id}
  • Delete User - HTTP GET /delete/{id}
